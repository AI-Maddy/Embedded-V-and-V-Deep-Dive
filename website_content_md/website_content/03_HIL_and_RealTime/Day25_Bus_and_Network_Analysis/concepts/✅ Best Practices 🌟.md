# ✅ Best Practices 🌟

\### 📝 Keep, Highlight, Update 📝

-   **Keep** one concept note per topic decision point for clarity and
    focus.
-   **Highlight** uncertainty and confidence explicitly to guide future
    reviews.
-   **Update** note whenever assumptions change to maintain relevance.
