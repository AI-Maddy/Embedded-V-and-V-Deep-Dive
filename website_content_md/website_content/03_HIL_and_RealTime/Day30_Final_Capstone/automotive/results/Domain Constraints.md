# Domain Constraints

☐ Compliance baseline: **ISO 26262 (ASIL) + ISO 21434** 📚 ☐ Key hazard
profile: unintended acceleration/deceleration, loss of stability,
braking faults 🚨 ☐ Interface landscape: CAN, LIN, FlexRay, Automotive
Ethernet 📈

Domain-Specific Mnemonic: **C.A.R.E.** 🚗 - **C**: Compliance (ISO
26262, ISO 21434) - **A**: Automotive (CAN, LIN, FlexRay, Automotive
Ethernet) - **R**: Requirements (hazard-linked, explicit, and
reviewable) - **E**: Evidence (nominal, boundary, fault, and residual
risk)
