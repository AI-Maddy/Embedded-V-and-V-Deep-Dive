# Best Practices 🌟

-   Tag every artifact with domain requirement IDs.
-   Capture timing + functional evidence in the same run package.
-   Record residual risk and ownership before closure.

::: admonition
Best Practice Reminder Always ensure traceability from requirements to
test results. This is a cornerstone of **DO-178C** and **IEC 62304**
compliance.
:::
