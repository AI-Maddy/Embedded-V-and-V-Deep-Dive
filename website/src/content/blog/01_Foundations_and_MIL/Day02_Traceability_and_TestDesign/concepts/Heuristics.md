# 🧪 Heuristics 🧪

-   **Measurability**: If it cannot be measured, it is not yet
    review-ready.
-   **Clarity**: If two reviewers interpret differently, refine wording.
-   **Failure Evidence**: If a failure is possible, define detection
    evidence.
