# Pitfalls ⚠️

::: important
::: title
Important
:::

Common pitfalls to avoid:

-   **Sensor/Actuator Fault Variants**: Missing test cases for sensor
    failures or actuator malfunctions.
-   **Weak Traceability**: Failing to link objectives to artifacts
    (e.g., test cases, logs).
-   **Configuration Drift**: Non-repeatable test reruns due to
    uncontrolled environment changes.
:::
