# GIVEN / WHEN / THEN Scenarios 🧩

**Nominal Scenario 🟢** GIVEN: A model with valid inputs and standard
operating conditions. WHEN: The simulation is executed. THEN: The output
matches the expected results within predefined thresholds.

**Boundary Scenario 🟡** GIVEN: A model with inputs approaching
operational limits (e.g., timing, value, or mode transitions). WHEN: The
simulation is executed under stressed conditions. THEN: The system
maintains functional correctness without exceeding limits.

**Fault Scenario 🔴** GIVEN: A model with invalid or unexpected inputs
simulating failure modes. WHEN: The simulation is executed. THEN: The
system detects the fault and executes recovery actions as specified by
requirements.
