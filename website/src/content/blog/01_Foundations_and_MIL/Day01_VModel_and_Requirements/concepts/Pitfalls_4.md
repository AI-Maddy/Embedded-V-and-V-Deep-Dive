# ⚠️ Pitfalls ⚠️

-   **Hidden model/configuration drift across runs**: Ensure model
    configurations are version-controlled and reproducible.
-   **Mixing requirement intent with implementation details**: Separate
    \"what the system should do\" from \"how the system achieves it.\"
-   **Missing traceability links in review notes**: Every verification
    step must trace back to a requirement or standard.
