# 📋 Pre-Closure Checklist

**Gate: Must be 100 % complete before fault injection evidence is
submitted**

-   [ ] All 5 verdict criteria (detection, latency, reconfig, safe
    state, recovery) evaluated.
-   [ ] Baseline (fault-free) comparison run exists for every fault

    run.

-   [ ] Fault coverage matrix reviewed by independent safety engineer.
-   [ ] Residual risks (uninjected FMEA items) justified and assigned to
