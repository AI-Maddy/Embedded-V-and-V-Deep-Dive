# Core Concept 🧠

-   Define the primary mechanism and expected behavior of the system
    under test.
-   Identify assumptions and boundary conditions that could affect
    system performance.
-   State what failure looks like and how it should be detected,
    ensuring clarity in the definition of success and failure criteria.
