# 🧪 Heuristics 🔬

-   One failing test without root cause is considered incomplete work;
    ensure thorough investigation.
-   Repeatability is required for confidence in test results; strive for
    consistent outcomes.
-   If evidence is missing, mark the result as provisional until all
    necessary data is gathered.
