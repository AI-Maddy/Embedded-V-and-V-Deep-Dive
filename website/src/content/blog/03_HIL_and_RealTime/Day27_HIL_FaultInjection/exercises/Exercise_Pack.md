# 📋 Exercise Pack

1.  **Nominal Scenario** 🟢: Execute a standard test case with explicit
    pass/fail criteria.
2.  **Boundary Scenario** 🟡: Test near operational limits (timing,
    value, or mode transitions) to assess system robustness.
3.  **Fault/Negative Scenario** 🔴: Simulate faults to evaluate
    detection and recovery behavior.
4.  **Rerun Consistency Check** 🟢: Verify repeatability of results
    under identical conditions.
