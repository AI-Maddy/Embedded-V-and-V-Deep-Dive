# Scenario Templates 📊

::: important
::: title
Important
:::

Use the following templates to document scenarios effectively.
:::

**GIVEN** a stable flight-control system, **WHEN** subjected to nominal
disturbances, **THEN** the system should maintain control authority. 🟢

**GIVEN** a high-workload transition near stability margins, **WHEN**
the system is tested, **THEN** it should remain within acceptable
performance thresholds. 🟡

**GIVEN** a corrupted bus label, **WHEN** the system experiences sensor
disagreement, **THEN** it should trigger appropriate fault management
protocols. 🔴
