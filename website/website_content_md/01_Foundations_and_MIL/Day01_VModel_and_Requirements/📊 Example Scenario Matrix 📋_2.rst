📊 Example Scenario Matrix 📋
-----------------------------
.. list-table:: MIL Scenario Matrix
   :header-rows: 1

   * - Scenario Type
     - Input Conditions
     - Expected Outcome
     - Evidence Artifact
   * - 🟢 Nominal
     - Valid inputs, nominal configuration
     - Outputs match requirements
     - Simulation logs, plots
   * - 🟡 Boundary
     - Edge-case inputs (e.g., max/min values)
     - Outputs within acceptable degraded range
     - Stress test logs, timing analysis
   * - 🔴 Fault
     - Invalid/noisy inputs
     - Fault detected, recovery triggered
     - Fault logs, error traces
